Technical Support / FAQ

System.Questions_and_Answers [v2.4]

[01]

How accurate is the Swift code generation?

Our export engine generates production-ready SwiftUI code with 99.9% layout parity. It maps blueprint constraints directly to Swift layout anchors and stacks.

[02]

Can I import existing Figma or Sketch designs?

Currently, Blueprint Gallery is a dedicated drafting tool focused on precision from scratch. We support PNG/SVG reference overlays to help you reconstruct designs with architectural accuracy.

[03]

Does it support collaborative real-time editing?

Yes. Multiple architects can work on the same .blueprint file simultaneously with sub-millisecond cursor synchronization and version branching.

[04]

What are the minimum system requirements?

Blueprint Gallery is optimized for macOS 13.0+ and iPadOS 16.0+. For the best experience, we recommend M1-series chips or newer for real-time physics snapping.
